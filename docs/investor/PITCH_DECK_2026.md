### Adaptive Emission Formula

```python
MintAmount = max(
    BaseReward × UtilityScore × QualityMultiplier × HalvingFactor,
    MinEmissionFloor × HalvingFactor
)

# Where:
# - UtilityScore: Network usage (0.0 - 1.0)
# - QualityMultiplier: AI validation quality (0.5 - 2.0)
# - HalvingFactor: Halves every ~4 years
```

### Why This Matters

| Scenario | Fixed Emission | Adaptive Emission |
|----------|----------------|-------------------|
| **Low Activity** | Waste supply → Inflation | Reduce emission → Preserve value |
| **High Activity** | Under-reward → Miners leave | Increase rewards → Attract miners |
| **Bear Market** | Same selling pressure | Lower emission → Less supply |

### Flow

1. **Miners** register AI models on subnets
2. **Validators** evaluate model quality through standardized tests
3. **Users** submit AI tasks and pay in MDT
4. **Network** distributes rewards based on validation scores
5. **Stakers** earn passive income from network fees

### Subnet Architecture
